# Religious Cognition Collaborative website

Static, responsive website configured for Vercel.

## Deploy with Vercel

1. Import `waypointinstitute/site`.
2. Set the Root Directory to `rcc-site`.
3. Select **Other** as the framework preset if Vercel does not detect it automatically.
4. Leave Build Command and Output Directory blank.
5. Deploy.

## Before final public launch

- Replace the temporary contact email if a general RCC address is created.
- Add the legal sponsoring nonprofit name and approved logo.
- Add approved current-project information and verified team members.
- Replace the initials portrait with an approved headshot.
- Add verified publications.
