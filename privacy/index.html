Privacy

Privacy

This page describes what information is and is not collected when you use this website.

What this website collects

This website is a static informational site. It does not set cookies, run analytics or advertising scripts, or track visitors. It does not include a newsletter, account system, or payment processing.

Contact forms and email

The forms on the Collaborate and Contact pages do not transmit information to a server. Submitting a form opens your own email application with a pre-addressed message; nothing is sent unless you choose to send that email. If you contact us by email, we use the information you provide only to read and respond to your message.

Hosting

This website is hosted on GitHub Pages. GitHub may collect technical information from visitors, such as IP addresses, for security and operational purposes. See the GitHub General Privacy Statement for details.

Changes

If the tools used by this website change—for example, if a newsletter, analytics, or donation system is added—this page will be updated to describe them before or when they are introduced.

Contact

Questions about this page can be sent to barrostheology@gmail.com.
