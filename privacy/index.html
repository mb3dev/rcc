What this website collects

This is a static informational website. It does not set cookies, run analytics or advertising scripts, or track visitors. It has no contact forms and collects no information through the site itself. If you email us, we use the information you provide only to read and respond.

This website is operated as part of the charitable and educational activities of True Dreams Project Fund, a U.S. 501(c)(3) nonprofit organization.

Hosting

This website is hosted on GitHub Pages. GitHub may collect technical information from visitors, such as IP addresses, for security and operational purposes. See the GitHub General Privacy Statement.

Changes

If the tools used by this website change—for example, if a hosted form, newsletter, analytics, or payment system is added—this page will be updated to describe them.
